class lampiapi.lampiapi.LampiAPI(logger: Logger | None = None)

Bases: object

A class for interacting with the Lampi API. ADMINAUTH and BACK_URL must be exposed as environment variables.

logger

The logger object used for logging.

Type:

Logger

Parameters:

logger (Logger, optional) – An optional logger object to use for logging. If not provided, a default logger will be created.

Environment Variables:

ADMINAUTH (str): The lampi API authentication secret.

BACK_URL (str): The lampi API url.

Example usage:

# Initialize the LampiAPI instance:

api = LampiAPI()

# Initialize with a custom logger:

custom_logger = MyLogger("custom.log", "custom_logger").get_logger()
api = LampiAPI(custom_logger)

class lampiapi.s3.S3(logger: Logger | None = None)

Bases: object

A class for interacting with Amazon S3 storage or any S3 compatible storage.

This class provides methods to perform various operations on S3, such as creating presigned URLs for file uploads and downloads, getting files from S3, deleting files, and listing files in a specified S3 bucket.

logger

The logger object used for logging.

Type:

Logger

Parameters:

logger (Logger, optional) – An optional logger object to use for logging. If not provided, a default logger will be created.

Environment Variables:

AWS_S3 (str): The value of the AWS_S3 environment variable (True for Amazon S3 False for a compatible S3).

AWS_PUBLIC (str): The value of the AWS_PUBLIC environment variable.

AWS_SECRET (str): The value of the AWS_SECRET environment variable.

DIRECTORY_PATH (str): The value of the DIRECTORY_PATH environment variable.

Example usage:

# Initialize the S3 client:

from lampiapi import S3
s3_client = S3()

# Create a presigned URL for uploading a file to S3:

upload_url = s3_client.create_presigned_post("my-bucket", "my-object")

# Create a presigned URL for downloading a file from S3:

download_url = s3_client.create_presigned_url("my-bucket", "my-object", organization_id, department_id, "pdf", 123)

# Get a file from S3 and save it locally:

s3_client.get_file("my-bucket", "my-object")

# Delete a file from S3:

deleted = s3_client.delete_files("my-bucket", "my-object")

# List files in an S3 bucket:

file_count = s3_client.list_files("my-bucket")
create_presigned_post(bucket_name: str, object_name: str, fields: dict | None = None, conditions: list | None = None, expiration: int = 60) → str

Generate a presigned POST URL for uploading a file to an Amazon S3 bucket.

Parameters:

  • bucket_name (str) – The name of the S3 bucket.

  • object_name (str) – The name of the object (file) to be uploaded to the bucket.

  • fields (dict, optional) – A dictionary of additional form fields to include in the presigned URL. These fields can be used to specify conditions for the upload. Default is None.

  • conditions (list, optional) – A list of conditions to apply to the presigned URL. Each condition is a dictionary with ‘acl’, ‘bucket’, ‘content-length-range’, or other S3 conditions as keys. Default is None.

  • expiration (int, optional) – The expiration time for the presigned URL in seconds. Default is 60 seconds.

Returns:

The generated presigned POST URL for uploading the file to S3.

Returns None if there was an error generating the URL.

Return type:

str or None

Example

# Generate a presigned POST URL for uploading a file to S3:

upload_url = s3_client.create_presigned_post(
    bucket_name="my-bucket",
    object_name="my-object",
    fields={"acl": "public-read"},
    expiration=3600,
)
create_presigned_url(bucket_name: str, object_name: str, organization_id: int, department_id: int, file_type: str, document_id: int, expiration: int = 3600) → str

Generate a presigned URL for downloading a file from an Amazon S3 bucket.

Parameters:

  • bucket_name (str) – The name of the S3 bucket.

  • object_name (str) – The name of the object (file) to be downloaded from the bucket.

  • organization_id (int) – The ID of the organization associated with the file.

  • department_id (int) – The ID of the department associated with the file.

  • file_type (str) – The type of the file (e.g., ‘pdf’, ‘jpg’).

  • document_id (int) – The unique ID of the document.

  • expiration (int, optional) – The expiration time for the presigned URL in seconds. Default is 3600 seconds (1 hour).

Returns:

The generated presigned URL for downloading the file from S3.

Return type:

str

Example

# Generate a presigned URL for downloading a PDF document from S3:

download_url = s3_client.create_presigned_url(
    bucket_name="my-bucket",
    object_name="example.pdf",
    organization_id=123,
    department_id=456,
    file_type="pdf",
    document_id=789,
    expiration=3600,
)
